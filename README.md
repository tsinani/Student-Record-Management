Student Record Management System Using Java 

This project is a simple Student Record Management System developed using Java. The purpose of this system is to manage student information in an organized way using Object-Oriented Programming concepts along with file handling techniques.
In this system users can add new students, search for a student by their ID, update existing student details, delete students, and display all stored student records.

The project also demonstrates different ways of storing data. Student records are saved in a text file for easy reading, a binary file for efficient storage, and also using object serialization to store complete objects.

In addition, the system generates a simple report that shows the total number of students, the highest GPA, the lowest GPA, and the average GPA using Java Streams.
The program also uses the File class to automatically create required folders and files, and it can display useful file information such as file name, path, size, and last modified date.

For safety, a backup system is included using buffered streams, which creates a copy of the student data file.

The project also includes proper exception handling to manage errors such as file not found or input/output problems.
Technologies used in this project include Java, Object-Oriented Programming, File Handling, Java Streams, Buffered Streams, and Exception Handling.

To run the program, open it in IntelliJ IDEA or any Java IDE and execute the main class. The system will automatically perform operations and generate required files.
